# sdwan-network-simulation
Features
	•	Routing and forwarding between networks
	•	Multi-node communication
	•	Basic network security rules

Concepts Used
	•	TCP/IP
	•	Routing
	•	Packet flow

Outcome

Gained understanding of SD-WAN architecture and network behavior.

## Simulation Logic
- Traffic types: video, voice, data
- Intelligent routing based on latency, bandwidth, and cost
- Dynamic path selection using Python logic

## Sample Output
Best link for video: Broadband  
Best link for voice: MPLS  
Best link for data: Broadband
